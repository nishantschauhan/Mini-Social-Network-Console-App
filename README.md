# Mini Social Network (C++)

A **console-based Mini Social Network** built in **C++**, allowing users to create profiles, send messages, and view inbox messages. All data is stored persistently in text files. This project demonstrates **OOP, file handling, and menu-driven interfaces** in C++.

---

## Features

- Create user profiles with unique usernames  
- Display all registered users  
- Send messages to other users  
- View received messages (Inbox)  
- Persistent storage using `users.txt` and `messages.txt`  

---

## Technologies Used

- **C++**  
- File handling (`fstream`)  
- Standard Template Library (STL): `vector`, `string`, `stringstream`  
- Console-based menu interface
